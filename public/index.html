Superconductor JS

Superfast data transfer, now available at room temperature.

Superconductor is a JavaScript library that allows you to transfer data between two (or more) devices at superfast speeds, using WebRTC, Sockets, utilizing P2P networks and a network of edge nodes. It's like a superfast, superconductive wire between two devices.
